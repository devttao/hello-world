# docker-nodejs

How to use
```shell script
docker login -u devttao -p dckr_xxxx

docker pull devttao/nodejs-kutao:latest
docker run -it -d -p 8081:80 --name nodejs-kutao_container devttao/nodejs-kutao:latest
curl localhost:8081
```
